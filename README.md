#  OrangeHRM Manual Testing Project

##  Project Overview

This repository contains detailed **manual test cases** for the OrangeHRM application.  
The project covers functional and UI/UX testing across multiple modules of the website.

The objective of this project is to ensure all modules function correctly, validate input handling, and verify system behavior under both positive and negative scenarios.
